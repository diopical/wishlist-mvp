# ⚡ Быстрый старт - Новые функции

## 🎯 За 5 минут!

Этот гайд поможет вам быстро начать использовать 3 основные новые функции.

---

## 1️⃣ Редактирование основной ссылки товара

**Зачем?** Исправить неправильную ссылку или заменить Amazon на Noon

**Как?**
1. Откройте вишлист для редактирования
2. Найдите товар → ищите кнопку ✏️ рядом со ссылкой
3. Нажмите ✏️ → появится поле для ввода
4. Введите новую ссылку (https://...)
5. Нажмите ✓ (сохранить изменение)
6. **Важно:** Нажмите "Сохранить все изменения" внизу страницы

```
Товар [img] | Название | ✏️ [Редактирование ссылки]
```

---

## 2️⃣ Добавление товара вручную

**Зачем?** Добавить товар, который приложение не может распарсить

**Как?**

### Способ А: Простой товар
1. Нажмите кнопку "✏️ Добавить товар вручную"
2. Заполните два обязательных поля:
   - **📝 Название:** "Чашка"
   - **🔗 Ссылка:** https://noon.com/... или любая другая
3. Нажмите "✓ Добавить товар"
4. ✅ Готово!

### Способ Б: С изображением (файл)
1. Нажмите "✏️ Добавить товар вручную"
2. Заполните название и ссылку (как выше)
3. Нажмите на кнопку "📁" для загрузки файла
4. Выберите изображение с компьютера
5. Нажмите кнопку ✓ рядом (подтверждение загрузки)
6. Нажмите "✓ Добавить товар"

### Способ В: С изображением (по URL)
1. Нажмите "✏️ Добавить товар вручную"
2. В поле "или по ссылке:" введите URL изображения:
   ```
   https://images.unsplash.com/photo-... (или любой другой URL)
   ```
3. Нажмите кнопку 📥 (загрузить)
4. После загрузки нажмите "✓ Добавить товар"

### Все поля формы:

```
📝 Название (обязательно)
💰 Цена (опционально) - "99.99 AED"
🔗 Ссылка (обязательно) - https://...
📸 Изображение (опционально)
💬 Комментарий (опционально)

[✓ Добавить товар] [Отмена]
```

## 3️⃣ Добавление комментария (пожелания)

**Зачем?** Указать пожелание по цвету, размеру или другие особенности

**Как?**

### На странице редактирования:
```
[Карточка товара]
  ↓
  [блок "💬 Комментарий"]
    ↓
    [✏️ Добавить]
      ↓
      [Текстовое поле]
        ↓
        [✓ Сохранить] [✕ Отмена]
          ↓
          [Требует: "Сохранить все изменения"]
```

**Пошагово:**
1. На странице редактирования найдите товар
2. Прокрутите вниз до коричневого блока "💬 Комментарий"
3. Нажмите кнопку "✏️ Добавить"
4. Впишите комментарий:
   - ❤️ "Хочу чёрный цвет"
   - 👕 "Размер M"
   - 🎨 "Больше никаких розовых"
5. Нажмите "✓ Сохранить"
6. Нажмите "Сохранить все изменения" внизу страницы

### На публичной странице:
Комментарий автоматически отобразится в оранжевом блоке перед кнопками:
```
[Товар]
[зарезервировано нет]
[💬 Пожелание: Хочу чёрный цвет]  ← Видно гостям!
[Ссылки магазинов]
[Зарезервировать]
```

## 🎯 Типовые сценарии

### Сценарий 1: Замена неправильной ссылки на Amazon
```
На странице редактирования вишлиста
→ Найду товар с неправильной ссылкой
→ Нажму ✏️ рядом со ссылкой
→ Введу правильный URL с Amazon
→ Нажму ✓
→ Нажму "Сохранить все изменения"
```

### Сценарий 2: Добавление товара с Noon
```
У меня есть товар на Noon, но его нельзя распарсить
→ Нажму "✏️ Добавить товар вручную"
→ Заполню:
   - Название: скопировано с Noon
   - Ссылка: ссылка с Noon
   - Цена: из Noon
   - Изображение: загружу с Noon
   - Комментарий: если есть пожелания
→ Нажму "✓ Добавить товар"
```

### Сценарий 3: Указание пожелания по цвету
```
Я добавил рубашку, но нужно указать цвет
→ Найду карточку товара
→ Нажму "✏️ Добавить" в блоке комментария
→ Напишу: "Хочу чёрный или темно-синий"
→ Нажму "✓ Сохранить"
→ Нажму "Сохранить все изменения"
→ Когда поделюсь вишлистом, гости увидят мое пожелание
```

## ⚠️ Важные моменты

| Что | Важно помнить |
|-----|---------------|
| **Ссылки** | 10 нужно нажать "Сохранить все изменения" внизу |
| **Комментарии** | Видны только на публичной странице, если их сохранить |
| **Изображения** | Хранятся как base64 (больше данных = дольше загрузка) |
| **Ручные товары** | Получают специальный ID `manual-{timestamp}` |
| **Альтернативные ссылки** | Нельзя напрямую редактировать, удалить можно через ✕ |

## 🆘 Быстрая справка по ошибкам

| Ошибка | Решение |
|--------|---------|
| "Ссылка не сохранилась" | Нажмите "Сохранить все изменения" внизу страницы |
| "Форма добавления не работает" | Заполните оба обязательных поля (название + ссылка) |
| "Комментарий не виден на публичной странице" | Обновите страницу (F5) - кэш браузера |
| "Изображение не загружается по URL" | Проверьте что это прямая ссылка на файл (.jpg/.png) |
| "Форма закрылась без сохранения" | Данные теряются, заполните форму заново |
